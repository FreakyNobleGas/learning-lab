# Tensorflow (TF)

- TF provides high-level APIs built on top of low-level APIs that provide more flexibility
- tf.keras is a high level API to define and train machine learning models to make predictions
    - It is the Tensorflow variant of the open-source Keras API (https://keras.io/)
